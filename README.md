# Dynamic Stories Studio

Dynamic Stories Studio is a modern multi-page photography portfolio website built using HTML, CSS, and JavaScript.

It is designed for photographers, creative studios, and visual storytellers who want a premium online presence.
---

## Features

- Full-screen cinematic hero section  
- Smooth scroll animations  
- Interactive portfolio with category filters  
- Masonry gallery layout  
- Lightbox image preview  
- Before & After comparison slider  
- About page with studio story and team section  
- Contact page with booking call-to-action  
- Custom cursor effects  
- Fully responsive design  

---

## Built With

- HTML5  
- CSS3  
- JavaScript  

---

## Project Structure

- index.html – Home page  
- portfolio.html – Portfolio gallery  
- about.html – Studio story & team  
- contact.html – Contact page  

---

## How to Run

1. Download the project  
2. Open `index.html` in your browser
